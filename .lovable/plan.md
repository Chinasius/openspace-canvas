

# OpenSpace — UI Engine by Olympus

Визуальный движок для создания интерфейсов с поддержкой XAML-разметки. Тёмный премиальный дизайн в стиле Olympus (чёрный фон, золотые акценты, звёздное небо).

---

## 1. Лендинг страница (Olympus стиль)
- Полноэкранная героическая секция с тёмным фоном, анимированными звёздами/частицами и логотипом "OpenSpace" в стилистике Olympus (золотой + мраморный текст)
- Описание проекта, команда Olympus
- Кнопка "Открыть редактор" — переход в рабочее пространство
- Секции с описанием возможностей движка

## 2. Редактор — основной интерфейс (десктопный стиль)
- **Верхняя панель (тулбар)**: логотип OpenSpace, меню (Файл, Редактировать, Вид), кнопки инструментов (курсор, перемещение, текст, фигуры)
- **Левая панель — Дерево элементов**: иерархия всех элементов на холсте (слои), drag&drop для изменения порядка, группировка элементов
- **Центральная область — Холст (Canvas)**: бесконечное рабочее пространство с сеткой, элементы можно перетаскивать, изменять размер, вращать; зум (колёсико мыши) и панорамирование
- **Правая панель — Свойства**: редактирование свойств выделенного элемента (размер, позиция, цвет, шрифт, border, opacity и т.д.)
- **Нижняя панель — XAML-редактор**: просмотр и редактирование XAML-разметки текущего проекта; двусторонняя синхронизация (изменил XAML → обновился холст и наоборот)

## 3. Доступные UI-элементы для размещения на холсте
- **Контейнеры**: Window (окно с заголовком, можно перемещать/закрывать), Panel, StackPanel, Grid
- **Базовые элементы**: Button, TextBlock, TextBox (input), Image, CheckBox, RadioButton
- **Расширенные**: Slider, ProgressBar, ListBox, ComboBox (dropdown), TabControl
- Каждый элемент имеет набор настраиваемых свойств (стили, размеры, привязки)

## 4. Ключевые фичи редактора
- **Drag & Drop** элементов из палитры на холст
- **Resize и Move** — изменение размера и позиции элементов мышкой с направляющими (snap-to-grid)
- **Многоуровневые слои** — Z-index, блокировка, скрытие элементов
- **Группировка** — объединение нескольких элементов в группу
- **Множественное выделение** (Shift+Click, область выделения)
- **Undo/Redo** (Ctrl+Z / Ctrl+Shift+Z)
- **Копирование/вставка** элементов (Ctrl+C / Ctrl+V)
- **Горячие клавиши** для основных действий
- **Несколько страниц/сцен** — переключение между страницами проекта

## 5. XAML-движок
- Парсинг XAML в визуальные элементы на холсте
- Генерация XAML из визуальных элементов
- Подсветка синтаксиса XAML в нижней панели
- Валидация XAML с отображением ошибок

## 6. Сохранение и экспорт
- Автосохранение в localStorage
- Экспорт проекта как XAML-файл
- Импорт XAML-файла
- Экспорт холста как PNG-скриншот

## 7. Дизайн и UX
- Тёмная тема по умолчанию: чёрный (#0a0a0a) фон, золотые (#c9a84c) акценты, мраморные текстуры для заголовков
- Интерфейс в стиле десктопного приложения (оконная система, минимальные отступы)
- Плавные анимации и переходы
- Адаптивный, но оптимизирован для десктопа
